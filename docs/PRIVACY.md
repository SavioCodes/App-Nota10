# Privacy Policy (Nota10)

Last updated: February 21, 2026

## 1. Scope
This policy explains how Nota10 handles personal data in web and mobile apps.

## 2. Data Collected
- Account identifiers (for authentication)
- Profile data (name, email, login provider)
- Study content uploaded by the user (documents/images)
- Generated learning artifacts (summaries, flashcards, questions)
- Usage and subscription status

## 3. Purpose of Processing
- Provide authentication and account access
- Store and process study material
- Generate educational artifacts
- Enforce plan limits and billing status
- Improve stability, security, and abuse prevention

## 4. Storage and Security
- Data is stored in managed infrastructure with access controls
- Secrets are restricted to server-side environments
- Logs are redacted to avoid leaking credentials and tokens

## 5. Sharing
We do not sell personal data. Data may be processed by infrastructure providers
strictly to deliver product functionality (auth, database, storage, billing).

## 6. Data Retention
Data is retained while the account is active. If the user requests account deletion,
associated records are removed according to backend deletion routines.

## 7. User Rights
Users can request account deletion from the profile screen.

## 8. Contact
For privacy requests, contact the project maintainer via repository contact channel.
